Langkah menggunakan aplikasi

  1. 1. Tambahkan data buku

    Masukkan judul, penulis, kategori, dan kode unik untuk setiap buku.

  2. 2. Daftarkan anggota

    Input data anggota: nama, kontak, dan nomor anggota.

  3. 3. Buat transaksi

    Buka menu Transaksi untuk mencatat peminjaman/pengembalian, lalu tekan tombol Simpan.

  4. 4. Pantau daftar transaksi

    Gunakan Daftar Transaksi untuk melihat riwayat dan status setiap transaksi.
